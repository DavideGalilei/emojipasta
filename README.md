# Emojipasta

#### 👳😦This 🅾 library 🎩⏰ allows you 😳✋ to 💀🅰 create 🚨 emojipastas 😂 like ‼☹ this 😍😍 one.

## Installation
```bash
$ nimble install https://github.com/DavideGalilei/emojipasta
```

## Usage
```nim
import emojipasta

echo emojify("Hello World")
```
